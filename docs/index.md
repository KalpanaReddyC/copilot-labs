# 🤖 GitHub Copilot — Hands-on Labs
{: .fs-9 }

A single set of **language-neutral** labs and **guided demos** for GitHub Copilot.
Every demo ships its starter code **inline on the page** — copy it, paste it into a file,
and run. **No git clone required.**
{: .fs-6 .fw-300 }

[Browse the Demos](demos){: .btn .fs-5 .mb-4 .mb-md-0 }

## Who this is for

Built for **large, mixed-language cohorts** (hundreds of participants). Each person picks
**one** language — **C++, Python, C#, JavaScript, or Java** — and follows the exact same steps.
Copilot writes the code; you review, run, and verify.

## Two tracks

| Track | What it is | Best for |
|---|---|---|
| [🎯 Hands-on Labs](labs) | Six language-neutral labs. You describe intent, Copilot writes the code. | Everyone — this is the core |
| [🎬 Guided Demos](demos) | Four presenter-led walkthroughs with **complete runnable code embedded on the page**. | Live demos + self-paced practice |

## Before you start

1. Read the [**Setup guide**](setup) once — install one language toolchain and confirm Copilot is signed in.
2. Learn the [**Copilot modes cheat sheet**](setup#copilot-features-cheat-sheet) (ghost text, inline chat, Ask/Plan/Edit/Agent, NES).
3. Pick a track above and go.

{: .highlight }
> **Zero-gap promise:** every demo page contains the full starter code **and** a collapsible
> solution. You never need to clone the repository to run anything — just copy from the page.
